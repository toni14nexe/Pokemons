# Pokemons

This template should help get you started local Pokemon project.

# RUN PROJECT

## Project Installation

```sh
npm install
```

### Compile and Run Development

```sh
npm run dev
```

### Run JSON-Server

```sh
json-server --watch json-server/db.json
```

### Login data

You can sign up as new user or try test user:

```sh
username: test
password: test1234
```
